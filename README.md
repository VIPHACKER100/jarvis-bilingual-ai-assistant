## üöÄ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (Version 18 or later)
- [NPM](https://www.npmjs.com/) or [Yarn](https://yarnpkg.com/)
- A modern browser with Speech Recognition support (**Chrome or Edge recommended**)

### Installation

1. **Clone the project:**

   ```bash
   git clone https://github.com/VIPHACKER100/jarvis-bilingual-ai-assistant.git
   cd jarvis-bilingual-ai-assistant
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Set up environment variables:**
   Create a `.env.local` file and add your Gemini API Key (Required for conversational AI features):

   ```env
   GEMINI_API_KEY=your_api_key_here
   ```

4. **Run the development server:**

   ```bash
   npm run dev
   ```
