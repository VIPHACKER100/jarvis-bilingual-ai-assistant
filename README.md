Operating on the **VIPHACKER100 OS**, it supports both **English** and **Hindi**, providing a natural and human-like interface for users in mixed-language environments. Inspired by the legendary Stark Tech, it features a stunning, mobile-optimized cyberpunk UI powered by a core **Arc Reactor**.

## ðŸš€ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (Version 18 or later)
- [NPM](https://www.npmjs.com/) or [Yarn](https://yarnpkg.com/)
- A modern browser with Speech Recognition support (**Chrome or Edge recommended**)

### Installation

1. **Clone the project:**

   ```bash
   git clone https://github.com/VIPHACKER100/jarvis-bilingual-ai-assistant.git
   cd jarvis-bilingual-ai-assistant
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Set up environment variables:**
   Create a `.env.local` file and add your Gemini API Key (Required for conversational AI features):

   ```env
   VITE_GEMINI_API_KEY=your_api_key_here
   ```

4. **Run the development server:**

   ```bash
   npm run dev
   ```
